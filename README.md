# Deep-Learning-Praxis Aufgabe
Dieses Repository enthält die Lösungen zur Praxisaufgabe im Modul Deep Learning (WS 2025/2026).

## Inhalt
- **Regression**: Vorhersage von Ernteerträgen basierend auf Düngereinsatz und Niederschlag (`harvest.csv`).
- **Klassifikation**: Unterscheidung von Tierarten (Dünnschnabelmöwe vs. Dickhornschaf) anhand von Größe und Umfang (`animals.csv`).

## Struktur
- `p1_regression.py` / `p2_regression.py`: Datenanalyse und MLP-Regressionsmodell.
- `p1_classification.py` / `p2_classification.py`: Datenvisualisierung und MLP-Klassifikationsmodell.

## Anforderungen
- Python 3.x
- Benötigte Bibliotheken: `pandas`, `matplotlib`, `scikit-learn`, `tensorflow`
